# 🌊 Boisar Tourism Platform

A comprehensive digital platform promoting eco and cultural tourism in Boisar, Maharashtra, India.

## 🌟 Features

- **🎬 Photo Carousel Background**: Immersive coastal and nature landscape slideshow
- **🔐 Authentication**: Complete login/register system with JWT-based auth
- **📱 Responsive Design**: Mobile-first approach with smooth animations
- **🤖 AI Chatbot**: Interactive Boisar tourism assistant
- **🗺️ Interactive Maps**: Explore destinations with Google Maps integration
- **📊 User Dashboard**: Personalized trip planning and booking management
- **🔗 Blockchain Payments**: Secure crypto-based bookings via Ethereum
- **🎨 Modern UI/UX**: Beautiful animations, glassmorphism, and transitions

## 🚀 Live Demo

Visit: [https://boisar-tourism.vercel.app](https://boisar-tourism.vercel.app)

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Styling**: Custom CSS with animations and glassmorphism
- **Backend**: Node.js + Express.js
- **Database**: MongoDB (via Mongoose)
- **Authentication**: JWT tokens with localStorage
- **Icons**: Font Awesome 6
- **Fonts**: Google Fonts (Poppins)
- **Maps**: Google Maps JavaScript API

## 📱 Mobile Features

- Touch-friendly navigation with hamburger menu
- Optimized photo carousel backgrounds
- Smooth scroll animations
- Responsive grid layouts
- Mobile-friendly chatbot interface

## 🎯 Key Sections

1. **Hero Section**: Full-screen photo carousel with animated content overlay
2. **Eco Tourism**: Beaches, forts, waterfalls, and hill treks around Boisar
3. **Cultural Heritage**: Warli art, coastal village life, local cuisine, Ganesh Festival
4. **Explore Boisar**: Interactive Google Maps with destination markers
5. **Plan Trip**: Trip planner, accommodations, transport, and activities
6. **Blockchain**: Secure crypto payments and verified local guides
7. **Sustainable Tourism**: Eco-friendly practices for responsible travel
8. **Traveler Stories**: Real testimonials from visitors
9. **Local Art & Handicrafts**: Warli paintings, bamboo craft, clay pottery
10. **Festivals & Events**: Local festival calendar with upcoming events
11. **Adventure Activities**: Trekking, beach activities, waterfall exploration
12. **Contact**: Multi-channel contact options with form

## 🏖️ Destinations Covered

- **Kelva Beach** – Black sand, coconut trees, peaceful environment
- **Kaldurg Fort** – Trekking with breathtaking monsoon views
- **Dabhosa Waterfall** – Dense forest waterfall near Jawhar
- **Kohoj Hill** – Panoramic views and nature walks
- **Dahanu Beach** – Chikoo orchards and coastal relaxation
- **Tarapur Fort** – Historic seaside fort with ocean views

## 🔧 Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/boisar-tourism.git

# Navigate to directory
cd boisar-tourism

# Start local server
python -m http.server 8000

# Open in browser
http://localhost:8000
```
